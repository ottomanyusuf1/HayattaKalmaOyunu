INSTALLATION
-------------
I. IMPORT FILES INTO YOUR PROJCT
Go to: “Assets -> Import Package -> Custom Package...” in the Unity Menu and select the “Stone_Lanterns_ver1.x.unitypackage” file. This will open an import dialog box. Click the import button and all files will be imported into your project list under a folder called JTS_StoneLanterns.

II. ADD ART PREFABS TO YOUR SCENE
In the /PREFABS folder you will find all StoneLanterns as individual game prefabs which you can use in your scene.  Also included are a set of stakcable stone block platforms which you can use in conjunction the the Stone Lantern objects.
